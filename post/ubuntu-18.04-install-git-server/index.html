Ubuntu 18.04 Install Git Server

@Fandaqian · Jul 22, 2019 · 1 min read

阿里云Ubuntu18.04 LTS安装git服务。

安装Git

sudo apt install git

创建账户git

sudo adduser git

创建证书登录
cd /home/git/
mkdir .ssh
touch .ssh/authorized_keys
chmod 600 .ssh/authorized_keys
cat id_rsa.pub >> ~/.ssh/authorized_keys
初始化git仓库
cd /home/git
mkdir gitrepo.git

cd gitrepo.git
git init --bare gitrepo.git
Initialized empty Git repository in /home/git/gitrepo.git/

chown -R git:git gitrepo.git/
客户端操作
ssh-keygen -t RSA -C "fandaqian@xx.com"

git config --global user.name "fandaqian"
git config --global user.email "fandaqian@xx.com"
git常用命令
git clone git@xx.xx.xx.xx:/home/git/gitrepo.git

git add .

git commit -m "commit message."

git pull origin master

git push origin master
